Overview:
ExamClock.html is a single 20KB html file that can be downloaded to any device with a Chromium browser installed for offline use, to be displayed on large displays used in classrooms, Theatres, Halls, Auditoriums for displaying a large live digital/analogue clock, a static start and end time (plus an extra time end clock), along with the exam paper information and additional notes for Invigilator/Student use in exam conditions.

Instructions:
With the page loaded, you can click on each of the clocks, live, start, end and extra time clocks to adjust the displayed time using the +/- minute buttons that appear.

Click on each of the relevant text fields to enter exam paper information.

There are buttons along the bottom of the page.
"Fullscreen" to enter fullscreen mode.
"Extra Time" to show/hide the additional extra time clock, beneficial for making the start/end time clocks larger if the extra time clock is not required.

Page has a built-in keep screen awake function, preventing screen from going to sleep and locking user account for extended display times.

Tip:
As ExamClock works offline, you can copy this onto any old off-domain laptop with no internet connection, hooked upto a basic projector, so that if the laptop is only ever used once a year during exam periods etc, it doesn't require any touch time beforehand with updates etc if always kept offline. As this html is built with the most basic of HTML5 and Javascript, hopefully the built-in functions will never by broken by any future Chromium updates, making this a useful exam centre clock for many years to come, all at zero cost to the establishment and ultimately, to tax payers.
